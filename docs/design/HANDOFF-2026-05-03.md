## 제한사항 기록

- **SVG fallback**: draw.io CLI(`drawio` binary) 미설치 환경에서 SVG를 직접 작성. rounded connector, auto-routing 등 draw.io 고급 기능 미지원. draw.io Desktop App에서 `.drawio` 파일을 열어 re-export 시 개선 가능.
- **검증 스크립트 환경**: `scripts/verify-docs.py`는 Python 3.x + stdlib만 사용. 외부 패키지 설치 불필요.
- **커밋/푸시**: 사용자 승인 대기 중.
